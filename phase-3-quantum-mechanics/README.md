## Phase 3 Checkpoint (blank page)

1. State the Born rule; compute ⟨x⟩, Δx and check the uncertainty bound.
2. Solve the infinite well from scratch; derive the oscillator spectrum with ladder operators.
3. Compute [x̂,p̂] and derive the uncertainty relation from it.
4. Reproduce the hydrogen energies and write the spin operators.
5. Build symmetric/antisymmetric two-particle states and state the Pauli principle.
6. Compute a first-order perturbation shift; set up a variational estimate; state the WKB tunneling factor.
7. State Fermi's golden rule.
8. Transform an operator into the Heisenberg picture; show a(t) = a(0)e^{−iωt}.
9. Add two spin-½'s into singlet and triplet.
10. Write fermion creation/annihilation operators, their anticommutator, and a many-body state — and explain why anticommutation enforces Pauli.

**第 3 阶段检查点（空白页）**

1. 陈述玻恩定则；计算 ⟨x⟩、Δx 并验证不确定性界。
2. 从头求解无限深方势阱；用梯形算符推导谐振子的能谱。
3. 计算 [x̂,p̂] 并由此推导不确定性关系。
4. 重现氢原子能级并写出自旋算符。
5. 构造对称/反对称两粒子态并陈述泡利不相容原理。
6. 计算一阶微扰能量修正；建立变分估计；陈述 WKB 隧穿因子。
7. 陈述费米黄金定则。
8. 将算符变换到海森堡绘景；证明 a(t) = a(0)e^{−iωt}。
9. 将两个自旋-½ 合成为单态和三重态。
10. 写出费米子产生/湮灭算符、其反对易子及多体态——并解释反对易为何强制满足泡利原理。
