# Physics Study Notes

A self-study curriculum that builds from mathematical foundations all the way to
**superconductivity**, written in a consistent **Definition → Demonstration → Application**
format. Each phase is a directory; each module is its own file.

The through-line: every step is a prerequisite for the next. From the calculus and
differential equations of Phase 0 to the BCS gap equation of Phase 5, the notes trace
exactly how a metal becomes a superconductor. Modules marked ⭐ are load-bearing for
what follows; ⭐⭐ are pivotal.

## How to use these notes

Each module follows **Definition → Demonstration → Application**. Re-derive the ⭐
results by hand and pass each phase's blank-page checkpoint (at the bottom of every
phase README) before moving on. When Modules 3.2, 3.5, 3.11, and 3.12 feel automatic,
you're ready for the condensed-matter and superconductivity phases that pay them off.
